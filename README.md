# MANAS-Mental-Analysis-for-Nations-and-Societies
Project MANAS employs a cutting-edge machine learning model to predict Disability-Adjusted Life Years (DALYs) due to mental disorders. This innovation enables informed healthcare planning and policy formulation globally.

# Run the project

Download and open the folder.
In terminal:
1. Open Manas_main in Live Server
2. Open MANAS_React by a. npm install
3.                     b. npm run dev
4. Open MANAS_Model_API by main.py by uvicorn main:app --reload --port 5000
